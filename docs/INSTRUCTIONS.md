# /docs

This folder contains visual documentation for the dashboard.

## Files to add manually

### `data_model.png`
A screenshot of the Power BI data model view showing the four tables (Download, CARGO, RealEstateProjectWP_Transactional, WorkPackages_Referential) and their relationships.

**How to capture:** In Power BI Desktop → Model view → screenshot.
Make sure no sensitive column data is visible — only table names and relationship lines.

### `dashboard_preview.png`
A screenshot of the dashboard canvas.

**Options:**
- Use the actual dashboard with data (if your organisation permits sharing anonymised screenshots)
- Replace all data values with placeholder text before screenshotting
- Export a PDF from Power BI and crop to the canvas area

**Recommended minimum:** Show the filter panel, the risk matrix visual, and at least one gate-status table so the viewer understands the scope of the dashboard.
